## 概述

### 挑戰

使用者應該能夠：

- [x] 當點擊問題時顯示/隱藏答案
- [ ] 僅使用鍵盤導航來導航問題並顯示/隱藏答案
- [x] 根據他們設備的螢幕尺寸查看介面的最佳佈局
- [x] 在頁面上查看所有互動元素的懸停和聚焦狀態
